Universal Sentence Encoder

Interfaz tipo Q&A

Funcionamiento

🧠 Modelo Universal Sentence Encoder (USE)

Se utiliza el modelo pre-entrenado de TensorFlow que convierte texto en vectores de 512 dimensiones, capturando el significado semántico de las oraciones.

📝 El modelo se carga desde CDN y funciona completamente en el navegador

📚 Base de Conocimiento Pre-embebida

Se procesa una base de datos de texto predefinida, generando embeddings (representaciones vectoriales) para cada fragmento de información.

⚡ Los embeddings se calculan una sola vez al cargar la página

🔍 Sistema de Similitud Semántica

Cuando haces una pregunta, se calcula su embedding y se compara con todos los embeddings de la base usando similitud coseno para encontrar las respuestas más relevantes.

🎯 El sistema no busca palabras exactas, sino significado similar

📊 Ranking de Resultados

Los resultados se ordenan por puntuación de similitud (0-1), mostrando las respuestas más relevantes primero.

🔢 Solo se muestran resultados con similitud > 0.5
